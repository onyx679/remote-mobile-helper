# 远程控制手机项目迁移清单

## 结论

不要把当前整个文件夹直接搬走。当前目录里大约 760MB，其中绝大部分是历史照片、调试截图、识别结果和日志。迁移到另一台设备时，只需要带核心脚本、配置文档和少量空目录。

已删除的浏览器 DOM 辅助、句子填空/补全单词方案不再迁移。

## 当前需要保留的功能

1. ADB / scrcpy 连接手机
   - USB 连接
   - Wi-Fi 连接
   - Android 11+ 无线调试配对
   - 小窗口低码率 scrcpy 投屏

2. scrcpy 视频画面单词识别
   - 只看 scrcpy 手机画面，不读浏览器 DOM
   - 从 scrcpy 窗口抓帧
   - OCR 识别单词
   - 本地词库判断 YES / NO
   - 输出到 `latest-answer.txt` 和 `latest-result.json`

3. 手机拍照自动复制到 Windows 剪贴板
   - 监听手机相册新照片
   - 拉取到 `camera-inbox`
   - 自动写入 Windows 剪贴板
   - 需要时可旋转图片后再复制

4. 辅助工具
   - 检查手机连接状态
   - 手动复制已有照片到剪贴板
   - 手动调整/查看 OCR 识别框
   - 查看运行状态
   - 停止识别/停止照片监听

## 另一台电脑需要安装的东西

1. Windows PowerShell
   - Windows 自带即可。

2. scrcpy
   - 用来投屏，也会提供 `adb.exe`。
   - 推荐安装后确认：

```powershell
scrcpy --version
adb version
```

3. Tesseract OCR
   - 当前脚本默认路径是：

```text
C:\Program Files\Tesseract-OCR\tesseract.exe
```

   - 安装后确认：

```powershell
& "C:\Program Files\Tesseract-OCR\tesseract.exe" --version
```

4. Python 3.10+
   - 当前这台电脑用的是 Anaconda Python 3.13。
   - 普通 Python 也可以，只要装依赖：

```powershell
python -m pip install -r requirements.txt
```

## 迁移后的第一次启动顺序

1. 把核心文件复制到新电脑，例如：

```text
C:\Users\<你>\Documents\远程控制手机
```

2. 安装 scrcpy、Tesseract、Python 依赖。

3. 手机开启开发者选项和 USB 调试，先用 USB 连新电脑，并在手机上允许调试授权。

4. 检查设备：

```powershell
.\check-devices.ps1
```

5. 记下输出里的设备 serial。

   当前脚本里写死的旧 serial 是：

```text
adb-2252475e-baGT88._adb-tls-connect._tcp
```

   到新电脑后很可能会变。可以先直接带参数运行：

```powershell
.\start-scrcpy-small.ps1 -Serial <新serial>
.\watch-phone-answer-fast.ps1 -Serial <新serial>
.\start-photo-clipboard.ps1 -Serial <新serial>
```

   如果想继续双击 `.cmd` 一键启动，就把新 serial 写回这几个文件的默认值：

- `start-scrcpy-small.ps1`
- `watch-phone-answer-fast.ps1`
- `start-photo-clipboard.ps1`

6. 启动 scrcpy：

```powershell
.\start-scrcpy-small.ps1 -Serial <新serial>
```

7. 启动单词识别：

```powershell
.\watch-phone-answer-fast.ps1 -Serial <新serial>
```

8. 启动拍照复制剪贴板：

```powershell
.\start-photo-clipboard.ps1 -Serial <新serial>
```

9. 查看状态：

```cmd
status.cmd
```

## 建议后续补的迁移友好功能

1. 做一个 `config.ps1`
   - 把 serial、OCR crop、scrcpy 码率、Tesseract 路径集中到一个配置文件。
   - 这样换电脑只改一个地方。

2. 做一个 `make-portable-package.ps1`
   - 自动打包必须文件。
   - 自动排除历史照片、debug 截图、日志和运行结果。

3. 做一个 `setup-new-pc.ps1`
   - 检查 scrcpy、adb、Tesseract、Python 包是否安装。
   - 缺什么就提示装什么。

4. 启动脚本自动选择设备
   - 如果只连了一台手机，就不用手动改 serial。
   - 如果连了多台，再让用户选。

5. 给单词识别和照片剪贴板增加日志
   - 出错时能快速知道是 ADB 断了、scrcpy 没开、OCR 没装，还是识别框不对。
